# Image Recognizer 
This API can recognize insects namely: 
1. `aphids`
2. `fruit fly`
3. `caterpillar`
4. `colorado potato beetle`
5. `leaf footed bugs`
6. `grasshoper`
7. `thrips`
8. `mexican bean beetle`
9. `leaf miners`
10. `white flies`
11. `mealy bug`
12. `spider mites`
13. `stink bug` 
which can create a serious loss for farmers around the world.
